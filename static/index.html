📖 使用说明 ▼
📸 相册系统
- 返图区点击 ♥ 收藏图片到当前相册（自动隐藏）
- 从 Output 文件夹扫描时可跳过已在相册的图片
- 💡 双击相册名称可重命名
- 🗑 相册删除需点击5次确认
- 📥 「导入图片」从 Output 文件夹批量导入到相册

📦 批量功能
- 「☑ 全选」一键勾选/取消所有图片
- 「🗑 确认删除」勾选后批量移出相册（非文件删除）
- 「🚀 批量发送」勾选多张图 → 用它们的提示词生成
- 「🔍 反推生图」取已存提示词送入队列生图

🎨 ComfyUI 使用说明

🚀 首次配置
1️⃣ ComfyUI中导出工作流：Workflow → Export (API) → 保存JSON到 workflows/ 文件夹
2️⃣ 刷新页面 → 下拉选择工作流 → 🔗 正负绑定 指定CLIP节点 → 💾 保存
3️⃣ 设宽高/张数 → ⚙ 调采样器/步数/CFG → 🎭 选模型/权重

🎮 四种生图方式
🚀 普通生图 — 用当前面板选好的标签直接生图
   💡 勾选「🤖AI自动润色」可先发大模型润色再出图
🎲 随机生图 — 每次重新随机抽标签再出图，张张不同
   💡 张数由上方「张」控制，配合随机种子可复现
✏ 手写生图 — 直接手写正向提示词，勾选润色会先AI加工
🖼 加载图片 — 上传图片绑定LoadImage节点，支持拖拽
   🔥 反推生图：选图片 → AI视觉模型分析 → 自动反推提示词 → 发送生图
   📂 内置6套反推预设：通用/转动漫/转真人/转油画/转赛博/中文反推

🤖 AI润色说明
- 标签发送时自动带分类名 [人物主体] [发型与发色] 等，大模型理解更准确
- 支持 OpenAI / LM Studio / Ollama / 自定义API
- 反推生图需要视觉多模态模型（gpt-4o / qwen-vl / glm-4v）
- 📜 润色记录自动保存最近100条，翻页浏览，可重新发送

📦 队列 & 返图
- 排队生成 → 进度条 → 逐一出图 → 可终止/清空
- 返图缩略图横向滚动 → 点击放大 → 缩放/拖拽 → ⬇下载 → 📝查看提示词
- 手机端通过Flask代理访问ComfyUI图片

💡 技巧
- 🎲 关闭随机种子 = 同标签同参数出固定图（可复现）
- 📐 模板区 {tags} 放LoRA触发词/画风前缀，润色不会影响模板内容
- 🚫 负面预设自动注入，和面板负面标签叠加生效
- 🔄 空隔转换：标记直接拼接 / 空格转下划线 / 下划线转空格
- 📱 手机连同一WiFi访问 http://电脑IP:5802 即可使用

📸 相册系统
- ♥ 收藏到相册 → 返图区隐藏，刷新不丢失
- 🖼 相册管理：新建/重命名/删除，双击改名称
- 📥 导入图片：从 Output 文件夹批量导入到相册
- 🛠 批量功能：全选 / 批量删除 / 批量发送 / 反推生图
- 🗑 清空到回收站：只删返图区显示的非相册图片
