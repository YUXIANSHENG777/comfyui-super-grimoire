📌 这是什么？
─────────────────────────────────────
一个 AI 绘画提示词工具，帮你选标签、组合提示词、一键发送 ComfyUI 生图。
桌面浏览器 + 手机浏览器都能用，数据自动同步。


🖥 环境准备（必须先装 Python 3）

─────────────────────────────────────
本工具需要 Python 3 才能运行。
如果你已经装过 Python，跳过此节直接看「怎么启动」。

📥 下载 Python 3 安装包
─────────────────────
👉 官网下载：https://www.python.org/downloads/
   推荐版本：Python 3.10 ~ 3.12（任选一个）
   直接下载 Windows 版：https://www.python.org/ftp/python/3.12.9/python-3.12.9-amd64.exe

⚙ 安装步骤（非常重要！）
─────────────────────
1. 双击下载的 python-3.x.x-amd64.exe
2. ⚠️ 务必勾选底部的「Add Python to PATH」（把 Python 加入系统路径）
   >>> 这步不勾选，后面无法启动！<<<
3. 点击「Install Now」（立即安装）
4. 等待安装完成，点「Close」

✅ 验证是否装好
────────────────
1. 按 Win + R → 输入 cmd → 回车
2. 黑窗口输入：python --version
3. 如果显示 Python 3.x.x 就成功了，
   如果显示"不是内部命令"→ 重新安装并勾选 Add to PATH

📌 常见问题
──────────
Q: 我之前装过 Python 但启动不了？
A: 装的时候没勾 Add to PATH。重新运行安装程序，
   选「Modify」→ 勾上 Add to PATH → 下一步完成即可。

Q: 装 Python 对电脑有影响吗？
A: 零影响，纯绿色。Python 只是一个运行环境，
   不占后台不弹广告，几千万人都在用。


🖥 怎么启动？
─────────────────────────────────────
1. 双击 start.bat
2. 浏览器自动打开 http://127.0.0.1:5802
3. 搞定，开始用

📱 手机端：手机连同一 WiFi，浏览器访问 http://电脑IP:5802
  （电脑 IP 在启动窗口的日志里能看到）


🎯 三步出图
─────────────────────────────────────
① 左侧分类树 → 点子类 → 中间标签网格 → 点标签加入
② 右侧面板可调权重(0.5~2.0)、拖拽排序、锁定标签
③ 选标签 → 点下方 🚀 生图（需先配置 ComfyUI）


🔗 ComfyUI 配置（只需要配一次）
─────────────────────────────────────
1. 启动 ComfyUI（默认 http://127.0.0.1:8188）
2. 在 ComfyUI 中搭好工作流 → Workflow → Export(API) → 保存为 .json
3. 把 .json 文件放到项目的 workflows/ 文件夹
4. 回到页面，刷新，下拉选择工作流
5. 🔗 正负绑定 → 指定正面/负面提示词的 CLIP 节点 → 💾 保存
6. 设置宽高、张数 → 🚀 生图


🤖 AI 润色怎么用？
─────────────────────────────────────
1. 启动 LM Studio / Ollama 或配置 OpenAI API
2. 在 AI 润色区填 API 地址和模型名
3. 点 🤖 发送 → 大模型把标签转为自然语言提示词
4. 勾选「AI自动润色」→ 每次生图前自动润色
5. 💾/📂 预设按钮可保存/加载系统指令


🎮 四种生图方式
─────────────────────────────────────
🚀 普通生图    — 用当前选好的标签直接出图
🎲 随机生图    — 每次重新随机抽标签，张张不同
✏ 手写生图    — 直接手写提示词发送
🖼 加载图片    — 上传图片绑定 LoadImage 节点
   🔄 反推生图 — 选图→AI看图画→自动出图（需视觉模型）


🔄 反推生图怎么用？
─────────────────────────────────────
1. AI润色区配置一个视觉模型（如 gpt-4o / qwen-vl）
2. 点 🖼 加载图片 → 选图片
3. 拉到弹窗底部 → 选反推预设 → 点 🔄 反推并生图
4. 大模型分析图片 → 反推提示词 → 自动发送 ComfyUI
   📂 内置6套预设：通用/转动漫/转真人/转油画/转赛博/中文反推


📂 文件说明
─────────────────────────────────────
start.bat        双击启动
server.py        后端服务（不要动）
launch.py        启动器（不要动）
workflows/       你的 ComfyUI 工作流 .json 放这里
data/            标签库
static/          网页前端
user_data/       你的个人数据（配置/收藏/预设/历史）
   ├─ history/   生图记录
   ├─ presets/   预设组合
   ├─ llm_config.json     LLM 配置
   └─ sync_data.json     同步数据


💡 常用技巧
─────────────────────────────────────
• 在标签网格点 ➖ 加入负面标签，点 ➕ 加入正面标签
• 拖拽标签可排序，点 🔒 锁定标签（随机时不变）
• 右侧面板 🔄 空隔转换：调整标签之间的连接方式
• 模板区 {tags} 占位符可放 LoRA 触发词
• 🚫 负面预设自动注入每次生图
• 随机种子关闭 = 同标签出固定图（可复现）
• 📜 润色记录保存最近 100 条，点一下就重新发送
• 📱 手机端底部 Tab 切换标签/提示词/生图
• 🌐 顶部 English/中文 按钮切换界面语言
