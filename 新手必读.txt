📌 这是什么？
─────────────────────────────────────
一个 AI 绘画提示词工具，帮你选标签、组合提示词、一键发送 ComfyUI 生图。
桌面浏览器 + 手机浏览器都能用，数据自动同步。


🖥 怎么启动？（超简单）

─────────────────────────────────────
1. 双击 start.bat
2. 智能启动器自动检测 Python：
   - 系统有 Python 3.8+ → 直接用
   - 系统没有 Python → 自动下载便携版（8MB，仅首次）
   - 自动安装依赖
3. 浏览器自动打开 http://127.0.0.1:5802
4. 搞定，开始用

💡 首次启动约需 30 秒（仅下载一次），之后秒开。
   Ctrl+C 可安全关闭服务器。


🖥 手动安装 Python（可选，启动器已自动处理）

─────────────────────────────────────
如果你更喜欢用系统 Python，可以手动安装。
启动器会自动检测到已安装的 Python 并优先使用。

📥 下载 Python 3 安装包
─────────────────────
👉 官网下载：https://www.python.org/downloads/
   推荐版本：Python 3.10 ~ 3.12（最低 3.8）
   直接下载 Windows 版：https://www.python.org/ftp/python/3.12.9/python-3.12.9-amd64.exe

⚙ 安装步骤（非常重要！）
─────────────────────
1. 双击下载的 python-3.x.x-amd64.exe
2. ⚠️ 务必勾选底部的「Add Python to PATH」
3. 点击「Install Now」（立即安装）
4. 等待安装完成，点「Close」

✅ 验证是否装好
────────────────
按 Win + R → cmd → 输入 python --version → 显示版本号即可

📱 手机端：手机连同一 WiFi，浏览器访问 http://电脑IP:5802
  （电脑 IP 在启动窗口的日志里能看到）


🎯 三步出图
─────────────────────────────────────
① 左侧分类树 → 点子类 → 中间标签网格 → 点标签加入
② 右侧面板可调权重(0.5~2.0)、拖拽排序、锁定标签
③ 选标签 → 点下方 🚀 生图（需先配置 ComfyUI）


🔗 ComfyUI 配置（只需要配一次）
─────────────────────────────────────
1. 启动 ComfyUI（默认 http://127.0.0.1:8188）
2. 在 ComfyUI 中搭好工作流 → Workflow → Export(API) → 保存为 .json
3. 把 .json 文件放到项目的 workflows/ 文件夹
4. 回到页面，刷新，下拉选择工作流
5. 🔗 正负绑定 → 指定正面/负面提示词的 CLIP 节点 → 💾 保存
6. 设置宽高、张数 → 🚀 生图


🤖 AI 润色怎么用？
─────────────────────────────────────
1. 启动 LM Studio / Ollama 或配置 OpenAI API
2. 在 AI 润色区填 API 地址和模型名
3. 点 🤖 发送 → 大模型把标签转为自然语言提示词
4. 勾选「AI自动润色」→ 每次生图前自动润色
5. 💾/📂 预设按钮可保存/加载系统指令


🎮 四种生图方式
─────────────────────────────────────
🚀 普通生图    — 用当前选好的标签直接出图
🎲 随机生图    — 每次重新随机抽标签，张张不同
✏ 手写生图    — 直接手写提示词发送
🖼 加载图片    — 上传图片绑定 LoadImage 节点
   🔄 反推生图 — 选图→AI看图画→自动出图（需视觉模型）


🔄 反推生图怎么用？
─────────────────────────────────────
1. AI润色区配置一个视觉模型（如 gpt-4o / qwen-vl）
2. 点 🖼 加载图片 → 选图片
3. 拉到弹窗底部 → 选反推预设 → 点 🔄 反推并生图
4. 大模型分析图片 → 反推提示词 → 自动发送 ComfyUI
   📂 内置6套预设：通用/转动漫/转真人/转油画/转赛博/中文反推


📐 分辨率快速选择
─────────────────────────────────────
点 ComfyUI 区的 📐 按钮 → 弹窗选常用分辨率
分三栏：📱竖屏 / ⬜方图 / 🖥横屏
从 512px 到 2K(2560×1440) 都有，点一下自动填宽高


🔔 生图完成通知
─────────────────────────────────────
每张图生成完成后：
🎨 彩色提示条（成功绿 / 失败红）
🔊 提示音（ComfyUI 区勾选开关）
💻 桌面通知（切换标签页也能收到）
📊 进度条显示「#序号 步 x/20」真实步数


🖼 图片管理
─────────────────────────────────────
📂 output 按钮：绑定 ComfyUI 的 output 文件夹，一键加载全部图片
🖼 提示词查看：点击图片放大预览 → 底部 📝 按钮 → 自动从 PNG 文件读取提示词（无需 ComfyUI）
♥ 收藏到相册：返图区点 ♥ 收藏图片，自动隐藏，刷新不丢失
🔄 刷新按钮：重新应用相册隐藏，不用整页刷
📂 相册管理：点底部卡片打开 → 新建/重命名/删除相册
  📥 导入图片：从任意文件夹批量导入到相册
  🔍 搜索过滤 + 🔤 排序菜单：按文件名/添加顺序/总面积/宽度排序
  🖼 自由比例浏览：图片保持原始比例，宽屏弹窗
  ☑ 批量功能：全选 → 批量删除 / 批量发送 / 批量反推生图
🗑 删除：鼠标悬停图右上角→连点两下→移至回收站（可还原）
🗑 清空到回收站：一键清空返图区（连点5次确认，相册内图片跳过）

📂 文件说明
─────────────────────────────────────
start.bat        双击启动
server.py        后端服务（不要动）
launch.py        启动器（不要动）
workflows/       你的 ComfyUI 工作流 .json 放这里
data/            标签库
static/          网页前端
user_data/       你的个人数据（配置/收藏/预设/历史）
   ├─ history/   生图记录
   ├─ presets/   预设组合
   ├─ llm_config.json     LLM 配置
   └─ sync_data.json     同步数据


⚙ 界面设置（工具栏 ⚙ 按钮）
─────────────────────────────────────
🪟 全局透明度 / 模糊程度 / 各区域透明度
🌙 深色模式勾选切换浅色/深色主题
🖼 自定义背景图：选本地图片 → 缩放模式 → 拖拽位置
✨ 页面特效（点「▶ 页面特效」展开）：
  💧 水面涟漪点击：点击页面产生 4 层蓝色涟漪
  🕊 3D 群鸟飞舞：鸟群在背景盘旋（透明背景叠加）
  🌊 3D 波浪海面：波浪起伏，色盘自定义颜色
点「确认」保存，点「重置默认」恢复

💡 常用技巧
─────────────────────────────────────
• 在标签网格点 ➖ 加入负面标签，点 ➕ 加入正面标签
• 拖拽标签可排序，点 🔒 锁定标签（随机时不变）
• 右侧面板 🔄 空隔转换：调整标签之间的连接方式
• 模板区 {tags} 占位符可放 LoRA 触发词
• 🚫 负面预设自动注入每次生图
• 随机种子关闭 = 同标签出固定图（可复现）
• 📜 润色记录保存最近 100 条，点一下就重新发送
• 📱 手机端底部 Tab 切换标签/提示词/生图，生图 Tab 底部有返图区和相册管理按钮
• 🖼 手机端图片预览也支持点击 📝 查看提示词
• 🌐 顶部 English/中文 按钮切换界面语言
